# Doc Timer 🕒

**Doc Timer** is a minimalist, high-visibility Android application that functions as both a digital clock and a professional countdown timer. It is designed for environments where visibility and simplicity are key, such as medical offices, classrooms, or during presentations.

## ✨ Features

- **Dual Mode:** Seamlessly switch between a live digital clock and a countdown timer.
- **Precision Input:** Set timers using specific **Hours and Minutes**.
- **Pinch-to-Zoom:** Intuitively resize the timer digits by pinching the screen—perfect for visibility from a distance.
- **Immersive Full-Screen:** Hide all UI elements and system bars for a distraction-free experience.
- **Custom Color Picker:** Choose from multiple high-contrast colors (Red, Green, Blue, etc.) to suit your preference.
- **Persistent State:** Automatically saves your color, text size, and timer progress when the device is rotated or the app is minimized.

## 🚀 How to Run

1.  **Clone the project** to your local machine.
2.  Open the project in **Android Studio**.
3.  Ensure you have the latest **Android SDK** installed.
4.  Connect an Android device or start an emulator.
5.  Click **Run 'app'** to install and launch.

## 🛠️ Built With

- **Java** - Main programming language.
- **XML** - Layout and UI design.
- **Android Material Components** - For modern, clean UI elements.
- **ViewBinding & Insets** - For edge-to-edge display support.

## 📖 Usage

- **Scaling:** Use two fingers to pinch the timer text to make it larger or smaller.
- **Full Screen:** Tap the "Full Screen Mode" button. To exit, tap the small icon in the bottom right corner.
- **Timer:** Click "Set Timer", enter your desired duration, and press "Start".
- **Color:** Tap "Change Color" to open the palette and select a new text color.
